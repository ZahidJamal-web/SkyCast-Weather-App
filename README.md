# 🌤️ SkyCast - Premium Weather App

SkyCast is a modern and responsive weather application that provides real-time weather information with a clean glassmorphism interface. Built using HTML, CSS, and JavaScript, it fetches live weather data from the OpenWeatherMap API and displays current weather conditions, hourly forecasts, and a multi-day forecast in an elegant user interface.

---

## 🚀 Features

- 🌍 Search weather by city name
- 📍 Auto-detect current location
- 🌡️ Real-time temperature and weather conditions
- 🌤️ Dynamic weather icons
- 💧 Humidity information
- 🌬️ Wind speed
- 🌡️ Atmospheric pressure
- 👀 Visibility
- 🌅 Sunrise time
- ☀️ UV Index support
- 🕒 Hourly weather forecast
- 📅 Multi-day weather forecast
- 🌙 Dark & Light mode
- 🎨 Dynamic weather-based backgrounds
- 📱 Fully responsive design
- ✨ Modern Glassmorphism UI
- ⚡ Fast and lightweight

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/ZahidJamal-web/SkyCast-Weather-App.git
```

### Open the project

```bash
cd SkyCast-Weather-App
```

Open `index.html` in your browser.

---

## 🔑 API Setup

Create a free API key from OpenWeatherMap.

Replace the API key inside `script.js`.

```javascript
const API_KEY = "YOUR_API_KEY";
```
